# Nirvana Benchmarks

Public benchmarking examples for [Nirvana Cloud](https://nirvanalabs.io). This repository contains ready-to-run infrastructure-as-code examples for deploying popular benchmarks on Nirvana's cloud platform.

## Available Benchmarks

- **[clickbench-hardware](./clickbench-hardware)** - ClickBench hardware performance benchmark for database systems

## Getting Started

### Prerequisites

- [Terraform](https://developer.hashicorp.com/terraform/downloads) >= 1.0
- [Nirvana Cloud account](https://dashboard.nirvanalabs.io)
- Nirvana CLI configured with your credentials
